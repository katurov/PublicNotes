## Обзор имеющихся государственных программ

На государственном уровне Сербия не развивает полноценную космическую программу. Страна не обладает собственными космическими технологиями, такими как ракеты-носители или спутники, и не имеет национального космического агентства. Интересно, что Сербия, как правопреемник СФРЮ, не представлена в Комитете ООН по использованию космического пространства в мирных целях (COPUOS), хотя Югославия была его членом. Это указывает на низкий уровень вовлеченности страны в международные космические структуры. Среди стран бывшей Югославии в COPUOS сейчас участвуют только Словения и Хорватия (последняя как наблюдатель).

Тем не менее, в Сербии существуют инициативы на уровне неправительственных организаций и энтузиастов. Одной из заметных структур является SERBSPACE — сербская НПО, которая позиционирует себя как участник космической деятельности. Также SERBSPACE инициировала создание "дочерней" структуры в Черногории под названием MONTESPACE. Однако эти инициативы остаются скорее декларативными: практических результатов, таких как запуск спутников или разработка технологий, они не принесли. Уже несколько лет эти организации не проявляют активности даже в соцсетях.

Государственных космических агентств, подобных NASA, в мире насчитывается несколько десятков, таких агентств около 30–40. Однако лишь немногие из них обладают полным спектром возможностей, как у NASA, включая собственные пусковые мощности и программы пилотируемых полётов. Только [11 стран](https://ru.wikipedia.org/wiki/Хронология_первых_космических_запусков_по_странам) (США, Россия, Китай, Япония, Индия, Евросоюз, Великобритания, Южная Корея, Израиль, Иран, Украина) доказали способность запускать объекты на орбиту своими ракетами. Лишь три страны (США, Россия, Китай) имеют опыт пилотируемых полётов.

Но если у вас в стране нет своего Агентства, это не должно останавливать от участия в космической программе. Есть три основных направления:
* Анализ данных
* Работать со спутниками
* Создавать ракеты (космические аппараты)

Бонус трек: делать Космос популярным (это наша фанатская обязанность)

### Анализ данных

Немногие космические агентства предоставляют открытые API или наборы данных для научных исследований, разработчиков и общественности. NASA славится своей политикой открытого доступа к данным, [включая API и обширные датасеты](https://data.nasa.gov): снимки Земли, данные о космосе и астрономические архивы. Кроме них активно распространяет данные [ESA (Европейский союз)](https://s2e2.cosmos.esa.int/www/esaddp_applications/Search_for_datasets.html). Что там можно найти:
* NASA (США)  
  * Открытое API: Да, NASA предоставляет набор открытых API через портал data.nasa.gov. Примеры включают API для данных об астероидах (NEO), снимков Земли (Earth), ежедневных изображений от телескопов (APOD) и данных марсоходов.
  * Наборы данных: Огромные архивы доступны через платформы вроде Astrophysics Data System (ADS) и NASA Open Data Portal. Данные включают снимки, телеметрию, климатические модели и многое другое. Все это в общественном достоянии.
* ESA (Европейский союз)
  * Открытое API: Да, ESA предоставляет доступ к данным через платформу ESA Open Data Portal. API доступны для некоторых миссий, например, спутников Sentinel (Copernicus program).
  * Наборы данных: Широкий доступ к данным о Земле, космосе и планетах через архивы миссий (например, Hubble, Gaia, Rosetta). ESA активно поддерживает политику открытых данных, хотя часть информации требует регистрации.

#### Химический дата-сайенс

Сегодня уже обсуждали, как хемоинформатика помогает анализировать данные. Такие данные могут быть с телескопов и зондов, вроде "Джеймса Уэбба" или марсоходов. Спектры из космоса показывают, какие вещества есть на других планетах (вода, метан, аминокислоты). Хемоинформатика сопоставляет эти данные с базами химических соединений, чтобы понять, есть ли там жизнь.

**Крутые истории**:
* Марсианские органические молекулы: В 2018 году Curiosity обнаружил органические соединения (тиофены, бензолы) в марсианском грунте. Хемоинформатики применили данные из PDS для моделирования этих молекул с помощью программ вроде [RDKit](https://www.rdkit.org), чтобы понять, как они могли образоваться (биогенно или абиогенно). Это связывает химию Марса с поиском жизни.
* Экзопланетные биомаркеры: Используя NASA Exoplanet Archive, учёные анализировали спектры атмосферы планеты K2-18b, где нашли воду. Хемоинформатики применили алгоритмы кластеризации к спектральным данным, чтобы выделить потенциальные биомаркеры (например, фосфин, PH₃), сравнивая их с земными аналогами.

**Наборы данных**


Kevin M Jablonka из Берлинскго University of Jena создал впечатляющую подборку данных для самостоятельной работы "**[awesome-chemistry-datasets](https://github.com/kjappelbaum/awesome-chemistry-datasets)**". 

На PubMED есть свой большой набор научных данных: **[Freely available compound data sets and software tools for chemoinformatics and computational medicinal chemistry applications](https://pmc.ncbi.nlm.nih.gov/articles/PMC3782340/)**

**[NASA Exoplanet Archive](https://exoplanetarchive.ipac.caltech.edu)** Что это: База данных с информацией об экзопланетах, включая спектры их атмосфер, полученные с телескопов вроде "Кеплера" или "Джеймса Уэбба". Спектроскопические данные содержат информацию о химическом составе атмосфер (например, метан, углекислый газ, вода). Хемоинформатики могут использовать эти спектры для идентификации молекул и моделирования их свойств с помощью QSAR (количественных отношений структура-активность) или машинного обучения. _Анализ метана (CH₄) в атмосфере экзопланет. Хемоинформатики сопоставляют спектры с базами данных вроде PubChem, чтобы предсказать, какие ещё молекулы могут присутствовать, или моделируют устойчивость соединений в условиях космоса._
* [Пример данных таблицей](https://exoplanetarchive.ipac.caltech.edu/cgi-bin/atmospheres/nph-firefly?atmospheres)
* [Пример конкретных данных](https://exoplanetarchive.ipac.caltech.edu/overview/51%20Eri%20b#planet_51-Eri-b_collapsible)


**[Planetary Data System (PDS)](https://pds-imaging.jpl.nasa.gov/portal/)** Что это: Архив данных с миссий NASA (например, марсоходов Curiosity и Perseverance), включающий химический анализ грунта и атмосферы Марса. Данные о минералах, органических соединениях и газах (например, CO₂, CH₄) можно анализировать для поиска биомаркеров или разработки материалов для марсианских баз. _Хемоинформатики используют данные о следах органики (например, хлорбензола) с Curiosity, чтобы смоделировать возможные пребиотические молекулы и их стабильность в марсианских условиях. Это помогает в астрохимии и поиске жизни._

**[NASA Earth Data](https://www.earthdata.nasa.gov) (через [GES DISC](https://disc.gsfc.nasa.gov))** Что это: Датасеты о составе земной атмосферы, собранные спутниками (например, Aura или Terra), включая концентрации газов (озон, NO₂, CO). Эти данные полезны для моделирования химических реакций в атмосфере или разработки экологически чистых материалов. _Хемоинформатики могут взять данные об озоне (O₃) и смоделировать его взаимодействие с новыми катализаторами, используя молекулярные дескрипторы и машинное обучение, чтобы улучшить материалы для защиты от радиации._

**[Astrophysics Data System](https://ui.adsabs.harvard.edu) (ADS) и открытые спектры** Что это: Коллекция научных статей и спектров из космических миссий, включая данные о межзвёздной пыли и молекулах в космосе. Спектры содержат "отпечатки" молекул (например, H₂O, CO, NH₃), которые можно анализировать для построения баз данных или предсказания свойств. _Исследователи используют спектры аммиака (NH₃) из межзвёздного пространства, чтобы создать виртуальные библиотеки соединений и проверить их устойчивость в условиях низких температур и вакуума._

**[NASA GeneLab](https://osdr.nasa.gov/bio/repo/search?q=&data_source=cgene,alsda&data_type=study)** Что это: Датасет с биологическими данными из экспериментов в космосе (например, влияние микрогравитации на молекулы в клетках). Химический анализ метаболитов и белков помогает разрабатывать лекарства для астронавтов. _Хемоинформатики анализируют изменения в метаболизме аминокислот в условиях космоса, чтобы предсказать, какие молекулы могут стабилизировать здоровье в длительных полётах._

#### О внеземной жизни и взгляде на Землю

Интересный факт: в 2017м году [миссия "Кассини"](https://ru.wikipedia.org/wiki/Кассини_(космический_аппарат)) была завершена, аппарат намеряно "сожгли" в атмосфере. Причиной такого способа завершения миссии была объявлена необходимость предотвратить заражение земными микроорганизмами спутников Сатурна, на которых не исключено наличие собственной жизни. На этом скрине - интерфейс управления миссии, видно, что это CDE вышедшая в 93м (миссия стартовала в 97, подумать только - 20 лет!). Common Desktop Environment (CDE) это среда рабочего стола, основанная на Motif, в основном использовалась в проприетарных операционных системах UNIX, OpenVMS. Система из времен, когда существовал промышленный стандарт рабочих мест UNIX 98, из эпохи, когда рабочие станции под управлением Unix властвовали в коммерческом сегменте.

Интереснейшая последняя фаза жизни аппарата, по итогам голосования среди посетителей сайта NASA получившая название «Большой финал» (Grand Finale), началась 23 апреля 2017 года. «Кассини» совершил 22 пролёта между Сатурном и его внутренним кольцом, которые позволили астрономам взглянуть на Сатурн и его спутники с новых ракурсов. 

Снимки "Кассини" доступны в библиотеке NASA, а я хочу обратить ваше внимание на [другое API](https://epic.gsfc.nasa.gov/about/api), позволяющее взглянуть на Землю с нового ракурса: на расстоянии в полтора миллиона киллометров (точка Лагранжа L1 системы Земля-Солнце) с 2015 года есть спутник-обсерватория "DSCOVR" (Deep Space Climate Observatory). Новые полноцветные фотографии полного диска Земли там появляются по нескольку раз в день, а история доступна за несколько лет (фото в размере 2048x2048 точек). Для своего [короткого видео](https://disk.yandex.ru/i/X8ihYlEdb6dItw) в 2017м году я склеил данные за 10 дней.

### Работа со спутниками

ПС-1 (простой спутник 1) был запущен 4 октября 1957 года (и проработал до 4 января 1958). Уже в 1985м году в "Гостье из Будущего" показывали, как школьники запускают свой спутник на орбиту в 2084 году (по понятной причине, внешне он был похож на ПС-1). Сейчас спутники действительно запускают студенты и школьники, а получение и передача через них информации доступно любому. Крупнейший проект по приему и передаче сообщений через спутники - [TinyGS](https://harmvandenbrink.medium.com/tracking-and-receiving-messages-from-satellites-with-lora-afafd875e005), в качестве транспорта используется LoRa. Список спутников по-настоящему впечатляет ([TinyGS, the Open Source Global Satellite Network](https://tinygs.com/satellites)) - можно посмотреть и частотные характеристики, и провести даже какой-то подсчет по странам. Есть одна в Белграде и Обреноваце, в Новом Саде - нет.

Эти спутники - часть программы CubeSat, появившейся в 1999м году, примерно в то же время на орбите начали эксплуатировать МКС. Примечательно, что на МКС собрано огромное количество приемо-передающей аппаратуры для комьюнити радиолюбителей. Например, используя рацию и сравнительно небольшую антенну, [можно позвонить на МКС](https://youtu.be/xLt5Vvgc1vA?si=POTlfNQJ5BChEt9J). Для такого сеанса потребуется лицензия (я обязан это упомянуть), так же, как для отправки пейджингового сообщения на МКС, а вот для приема таких сообщений никакой лицензии не потребуется, более того, вам хватит простого [Commodore 64](http://c64-iss.ddns.net) ([YT](https://youtu.be/WJVU1stPPlQ?si=pCFhcplaW2CzwpWp))!

Если хочется чего-то более практичного, можно [наладить связь с другом в Арктике](https://www.youtube.com/watch?v=JbXaBDRQ-q0&list=PL3XBzmAj53Rk0AG8yK06CZU1LeSDxnN-9), для чего придется поискать глобальный синхронизирующий сигнал, где на помощь вам придет GPS. Очень удобно с его стороны. Но это на нашей планете, а что делать, если ваш друг на Марсе?

### Sonic Pi

[Официальный сайт](https://sonic-pi.net) Sonic PI

Для начала, объединяя тему музыки и космоса, можно / нужно проиграть музыку из «Близкие контакты третьей степени» (англ. Close Encounters of the Third Kind, 77)
```ruby
# Космическая версия темы из "Близкие контакты третьего рода"
use_bpm 120  # Устанавливаем темп (beats per minute)

# Определяем мелодию как массив нот
theme = [:g4, :a4, :f4, :f3, :c4]

# Функция для проигрывания темы
define :play_theme do
  use_synth :tri  # Треугольный синтезатор для "инопланетного" звука
  theme.each do |note|
    play note, release: 0.6, cutoff: 80  # Добавляем фильтр для мягкости
    sleep 0.5
  end
end

# Живой цикл с эффектами
live_loop :alien_signal do
  with_fx :reverb, room: 0.8 do  # Реверб для "космической глубины"
    with_fx :echo, phase: 0.25, decay: 2 do  # Эхо для "сигнала издалека"
      play_theme
    end
  end
  sleep 2  # Пауза перед повтором
end

# Добавляем фоновый "космический шум"
live_loop :space_noise, sync: :alien_signal do
  use_synth :noise
  play 60, release: 4, amp: 0.2, cutoff: rrand(50, 80)  # Случайный фильтр
  sleep 4
end
```
И для завершения доклада, с упоминанием HAL9000 ([который можно собрать самому](https://learn.adafruit.com/hal-9000-replica?view=all))
```ruby
use_bpm 100  # Чуть медленнее, чтобы звучало драматично

# Мелодия "Daisy, Daisy"
daisy = [:f4, :f4, :a4, :bb4, :a4, :g4, :f4, :g4, :a4, :f4, :g4, :f4, :e4, :f4]

# Функция для проигрывания с замедлением
define :play_daisy do |speed|
  use_synth :saw  # Пилообразный синтезатор для "электронного" голоса HAL
  daisy.each_with_index do |note, i|
    play note, release: 0.8, cutoff: 60 - (i * 2)  # Постепенно снижаем частоту
    sleep speed * (1 + i * 0.1)  # Увеличиваем паузу, как будто HAL "зависает"
  end
end

# Добавляем эффекты и проигрываем
with_fx :reverb, room: 0.9 do  # Реверб для "космической пустоты"
  with_fx :distortion, distort: 0.3 do  # Искажение для "поломки"
    play_daisy 0.5  # Начальный темп
  end
end

# Финальное "затухание" HAL
sleep 2
play :f4, release: 2, amp: 0.5, cutoff: 40  # Последняя нота, как "прощание"
```

### E.T. Call Home

[Texas Instruments LPC Speech Chips](https://en.wikipedia.org/wiki/Texas_Instruments_LPC_Speech_Chips) - The Texas Instruments LPC Speech Chips are a series of speech synthesizer digital signal processor integrated circuits created by Texas Instruments beginning in 1978. They continued to be developed and marketed for many years, though the speech department moved around several times within TI until finally dissolving in late 2001.

Удивительным образом, у этого есть современная [армия фанатов](https://www.vintagecomputing.com/index.php/archives/528/vcg-interview-richard-wiggins-talks-speak-spell) с довольно [яркими проектами](http://furrtek.free.fr/index.php?a=speakandspell&ss=1&i=2).
